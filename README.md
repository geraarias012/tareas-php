# Sistema de Tareas (CRUD)

Aplicación web desarrollada en PHP y MySQL que permite gestionar tareas.

## Funcionalidades

- Crear tareas
- Listar tareas
- Editar tareas
- Eliminar tareas

## Tecnologías

- PHP
- MySQL
- HTML

## Cómo ejecutar

1. Clonar repositorio
2. Importar database.sql
3. Ejecutar en XAMPP
